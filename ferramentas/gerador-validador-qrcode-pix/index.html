O Que é QR Code Pix (BR Code)?

BR Code é o nome técnico do padrão por trás de todo QR Code Pix e do "Pix Copia e Cola". Definido pelo Banco Central com base no padrão internacional EMV Merchant-Presented QR Code, é uma sequência de texto estruturada em campos (chave Pix, nome do recebedor, cidade, valor, identificador da transação) terminada por um código de verificação de integridade.

Quando você aponta a câmera para um QR Code Pix, o app do banco decodifica exatamente esse texto — não existe "mágica" na imagem, ela apenas carrega essa string de forma compacta e legível por câmera. É por isso que o mesmo pagamento pode ser feito colando o texto diretamente ("Copia e Cola"), sem escanear nada.

O Golpe do QR Code Pix Adulterado

Como o BR Code é apenas texto plano, alterar a chave Pix dentro dele é trivial para quem tem as ferramentas certas. Esse é o princípio por trás de golpes reais e recorrentes:

  • Troca física do adesivo — golpistas colam um adesivo com QR Code próprio por cima do QR Code legítimo em estabelecimentos, comércios de rua e até em boletos/carnês impressos.
  • BR Code adulterado por mensagem — o texto "Copia e Cola" é reenviado por WhatsApp ou e-mail com a chave Pix trocada, geralmente em golpes de falso vendedor ou falso boleto.
  • Phishing com QR Code — imagens de QR Code em sites falsos ou anúncios fraudulentos redirecionam o pagamento para uma conta controlada pelo golpista.
O que o CRC16 detecta — e o que não detecta

O CRC16 ao final do BR Code é um checksum de integridade do texto, não uma assinatura digital. Ele confirma que o código não foi corrompido ou truncado após ser gerado (ex: erro de cópia, corte de caracteres). Se um golpista gera um BR Code do zero com a própria chave Pix, o CRC dele estará matematicamente correto — o código "passa" na validação porque é internamente consistente, só que aponta para a conta errada. Use a validação de integridade combinada com a conferência visual do nome do recebedor e da chave Pix exibidos pelo app do banco antes de confirmar qualquer pagamento.

Como Funciona a Verificação de Integridade (CRC16)

O BR Code é montado em campos no formato TLV (Tag-Length-Value): um identificador de 2 dígitos, o tamanho do valor em 2 dígitos, e o valor em si. Os principais campos são o GUI fixo br.gov.bcb.pix, a chave Pix, o valor (opcional), o nome e a cidade do recebedor.

Ao final de tudo, é calculado um CRC-16/CCITT-FALSE (polinômio 0x1021) sobre o texto completo do código. Esse valor de 4 dígitos hexadecimais é anexado ao BR Code. Ao validar um código, esta ferramenta recalcula o CRC16 sobre o texto recebido e compara com o valor informado: se não baterem, o código foi alterado, truncado ou corrompido em algum ponto após a geração.

Perguntas Frequentes

O que é QR Code Pix (BR Code)?

BR Code é o nome técnico do padrão usado nos QR Codes e no Pix Copia e Cola. É uma sequência de texto no formato EMV Merchant-Presented que carrega a chave Pix do recebedor, nome, cidade, valor opcional e um código de verificação (CRC16) ao final.

Como funciona o golpe do QR Code Pix adulterado?

Golpistas trocam fisicamente o adesivo do QR Code em estabelecimentos, ou enviam um BR Code manipulado por WhatsApp/e-mail, redirecionando o pagamento para uma chave Pix diferente da do recebedor legítimo. Como o BR Code é apenas texto, editar a chave Pix dentro dele é trivial.

O que o CRC16 verifica exatamente?

Detecta corrupção ou adulteração acidental no texto do código — se um caractere for alterado, o CRC recalculado não bate mais. Não impede o golpe de troca de chave Pix desde a origem, pois um código gerado do zero pelo golpista terá CRC correto para os dados errados que ele escolheu.

Meus dados são enviados para algum servidor?

Não. A geração e a decodificação rodam 100% no seu navegador via JavaScript. Nenhuma chave Pix, valor ou BR Code colado é enviado aos nossos servidores ou a terceiros.

Por que nome e cidade aparecem em maiúsculas sem acento?

O padrão BR Code do Banco Central exige texto ASCII nesses campos. A ferramenta normaliza automaticamente (remove acentos e converte para maiúsculas) antes de montar o código, seguindo a mesma regra aplicada por bancos e adquirentes.

Esta ferramenta gera QR Code Pix real para receber pagamentos?

Sim, o texto Pix Copia e Cola gerado segue o padrão oficial e pode ser usado em apps bancários. Esta ferramenta gera apenas o texto — não renderiza a imagem do QR Code. Sempre confira os dados antes de compartilhar ou pagar.
